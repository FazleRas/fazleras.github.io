AlphaLab

Live demo Code

Full-stack trading analytics platform: real-time quotes, technical indicators computed from scratch in pandas, a multi-ticker signal scanner, and a multi-strategy backtesting engine with equity curves, benchmark overlays, and risk metrics (CAGR, Sharpe ratio, max drawdown).

  • Parameter sweeps run an RSI-threshold grid in one request and render it as a heatmap — with one-click out-of-sample validation: optimize on the first 70% of history, then re-test the winners blind on the held-out 30%.
  • Per-user watchlists and saved backtests behind Supabase auth — JWT verification in FastAPI plus Postgres row-level security as defense-in-depth.
  • Optional Redis market-data cache in front of yfinance that fails open when Redis is down and reports hit/miss stats.
  • Deployed and live: React + Tailwind frontend on Vercel, Dockerized FastAPI backend on Render.

ACROBOT

Code

AI Code Review Org Bot — a GitHub Action that reviews pull-request diffs with an LLM and posts inline comments, engineered so that being wrong is cheap and being spammy is impossible.

  • Treats the LLM as untrusted input: output is schema-validated JSON, and every finding's line anchor is verified against the parsed diff before anything is posted.
  • Built around a hard constraint — the Gemini free tier. A two-clock rate limiter (requests-per-minute window + daily budget) degrades to a partial review instead of failing CI when the budget runs out.
  • Content-fingerprinted comments make re-runs idempotent across force-pushes; 49 tests plus a precision/recall eval harness.
  • Dogfooded on AlphaLab: the bot reviews every pull request to the platform above. Day one it caught 2/2 deliberately planted bugs with exact counterexamples.
